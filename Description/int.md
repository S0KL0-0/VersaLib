# Int Functions

- [Round](#round)
- [Factorial](#factorial)
- [Reverse](#reverse)

## Round

Convert a float to int.

`round(float)`

```cpp
float example = 19.325;
cout << round(example);
```
```cpp
Output: 19
```

## Factorial

Returns a factorial of a intiger.
Returns a long long.

`factorial(int)`

```cpp
int example = 12;
cout << factorial(example);
```
```cpp
Output: 479001600
```

## Reverse

Reverses an intiger

`reverse(int)`

```cpp
int example = 12345;
cout << reverse(example);
```
```cpp
Output: 54321
```
