### Table of Contents

# Round

Convert a float to int.

**round(float)**

```
float example = 19.325;
cout << round(example);
```
```
Output: 19
```

# Factorial

Returns a factorial of a intiger.
Returns a long long.

**factorial(int)**

```
int example = 12;
cout << factorial(example);
```
```
Output: 479001600
```
